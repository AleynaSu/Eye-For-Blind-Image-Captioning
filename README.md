# Eye-For-Blind-Image-Captioning
An AI-powered image captioning system designed as an assistive technology for visually impaired individuals, featuring an 'Assistant Voice' output.
Bu çalışma, derin öğrenme (Deep Learning) tekniklerini kullanarak görselleri anlamlandıran ve görme engelli bireyler için sesli betimleme sağlayan bir asistan uygulamasıdır.
